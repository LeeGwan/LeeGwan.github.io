## Hello

This is the first post. Each post is a single Markdown file in the `_en`
folder (English) or `_ko` folder (Korean). When you push it to GitHub, the
site rebuilds automatically.

## Front matter

Every post starts with a small header between `---` lines:

```yaml
---
title: "My post title"
date: 2026-05-24
category: "Windows Internals"   # big category (shown in top Posts menu)
group: "Overview"               # subsection (shown in the left sidebar)
weight: 2                        # sort order inside the group
ref: my-post                     # SAME value in the EN and KO versions
tags: [Windows, Reversing]
summary: "One-line description."
---
```

`category` is the big bucket in the top **Posts** menu. `group` is the
subsection inside the left sidebar. `ref` links the EN/KO versions together.

## Writing the body

Just write Markdown. Inline code looks like `IoAllocateMdl`, and fenced blocks
get syntax highlighting automatically.

## Table of contents

The two lines `* TOC` and `{:toc}` at the top auto-generate the box of links
you see above, built from your `##` headings.
